About

I was 'proficient' in Microsoft FrontPage (RIP), and had a GeoCities website or ten back in the early-mid 2000's. Like most people, I modified my MySpace page in a very tasteful manner! My biggest achievement was when the design of my personal website using chromeless windows got me invited to host on a private domain. They were all the rage back then.


I then did nothing for 15+ years, until March 2018 when I got put in charge of a small (offline) business website, using Wordpress. In December 2020 (mid-pandemic) I started coding again from scratch!


I'm currently working on DOM manipulation with JavaScript, and learning about array methods and objects. I'd really like to learn more about CSS animation, JavaScript and CSS libraries and frameworks, accessibility and UX and UI design.
